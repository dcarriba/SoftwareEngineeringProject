# SoftwareEngineeringProject

Project for the Software Engineering course at Université Côte d'Azur.

## Author

Daniel Carriba Nosrati

## Requirements

Java is required. Java version 21 is recommended.

## Instructions to build and run

### Build the project

```bash
./gradlew build
```

### Run the project

```bash
./gradlew run
```

### Run unit tests

```bash
./gradlew test
```
